# 🏔️ Minimal LLM Workspace
> **Extreme Speed. Absolute Privacy. Minimalist Elegance.**

A clean, high-performance AI workspace built with Next.js. Bring your own keys and control your data.

## ✨ Features
- **BYOK (Bring Your Own Key)**: Support for OpenAI, Anthropic (Claude), and Google (Gemini).
- **Infinite History**: Local persistence via IndexedDB (Privacy First).
- **Cloud Sync**: Optional cloud-sync via Supabase for multi-device support.
- **Minimalist Aesthetic**: Apple-inspired design with Pure Black, Glassmorphism, and Paper themes.
- **Fast Messaging**: Instant rendering with atomic session updates.
- **File Support**: Attach images and documents for deep analysis.

## 🚀 Quick Start
1. **Deploy to Vercel**: Hit the deploy button on your fork.
2. **Setup Keys**: Open settings in the app and paste your API keys.
3. **Enjoy**: Type your directive and get instant results.
